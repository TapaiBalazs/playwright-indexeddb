How this auto-increment queue works

This showcase demonstrates an IndexedDB object store that assigns numeric keys automatically as items are added.

Use this page in the following way:

  1. Type a value into the queue input and submit the form. Each submitted value is stored as a new IndexedDB record.
  2. Review the table below the form. It lists each stored record with its generated key and saved value.
  3. Use Delete first to remove the earliest record in the queue.
  4. Use Delete last to remove the most recently stored record.
  5. Use Clear All to remove every record one by one while showing a status message for each deleted entry.
  6. Use Delete database to remove the full IndexedDB database for this example.

This example is useful when you need a lightweight queue or append-only list where the browser assigns stable record identifiers.
